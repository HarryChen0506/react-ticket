### 前端初始化
```
 //启动前端
 cd /
 $ npm install
 $ npm run start

 //build
 $ npm run build
```
### 后端初始化
```
 //启动后端 (注意，要先安装mongodb数据库以及启动数据库)
 $ cd server
 $ npm install
 $ npm run server
```
